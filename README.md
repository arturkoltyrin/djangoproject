# Django Project (интернет магазин)

## Описание проекта

Этот проект представляет собой веб-приложение для интернет-магазина, разработанное с использованием Django. 
Он включает в себя приложение `catalog`, которое содержит маршруты для домашней страницы и страницы
с контактной информацией. Для стилизации страниц используется Bootstrap.

## Установка
1. Клонируйте репозиторий:
git clone https://github.com/arturkoltyrin/djangoproject.git

2. Создайте и активируйте виртуальное окружение:
python -m venv venv

3. Установите зависимости:
Убедитесь, что у вас установлен Poetry.
Затем выполните команду: poetry install

4. Запустите сервер разработки:
python manage.py runserver

## Описание функций

### Приложение `catalog`
- **views.py**
  - `home(request)`: Отображает домашнюю страницу, используя шаблон `home.html`.
  - `contacts(request)`: Отображает страницу с контактной информацией, используя шаблон `contacts.html`.

- **urls.py**
  - `path('home/', views.home, name='home')`: URL для доступа к домашней странице.
  - `path('contacts/', views.contacts, name='contacts')`: URL для доступа к странице с контактной информацией.

### Конфигурация проекта
- **settings.py**: Содержит стандартные настройки проекта Django, включая настройки базы данных, статических файлов и приложений.
- **urls.py**: Основной файл маршрутизации, который включает маршруты для приложения `catalog` и определяет URL.

## Использование

- Перейдите по адресу `http://127.0.0.1:8000/catalog/home/` для доступа к домашней странице.
- Перейдите по адресу `http://127.0.0.1:8000/catalog/contacts/` для доступа к странице с контактной информацией.

## Добавление новых приложений
- Создаем новое приложение:
python manage.py startapp
Зарегистрием приложение в settings.py, добавив его в список INSTALLED_APPS.
Настроем маршрутизацию для нового приложения, добавив соответствующие URL-адреса в urls.py.

## Проверка качества кода
Проект использует следующие инструменты:

- Flake8 для проверки стиля кода.
- Black для автоматического форматирования кода.
- isort для упорядочивания импортов.
- Mypy для статической проверки типов.
